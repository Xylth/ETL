#Projet ETL
ce projet permet d'extraire les données du site http://books.toscrape.com/

## Pré-requis 

-Python

### installation

-Télechargez le repo
-Lancez un terminal à la racine du dossier
-Créez un virtual environment nommé .env
-Activez le virtual environment
-Installez les dépendances dans requirements.txt

## Utilisation 
-Pendant que le virtual environment est activé à la racine du dossier, lancez la commande
''''
python -m app
'''' 
-Les données se téléchargeront dans le dossier exports à la racine du dossier
